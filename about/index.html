about
WhetKit.

A collection of privacy-first developer tools that run entirely in your browser. No data ever leaves your device. No servers, no uploads, no tracking.

why WhetKit?
>
100% private

All processing happens in your browser. We never see your data. Not a single byte leaves your machine.

>
no servers

Every tool is a static HTML file. There is no backend, no database, no API — just pure front-end code.

>
no tracking

We use GoatCounter for basic analytics — privacy-focused, no cookies, respects Do Not Track.

>
always free

Every tool is and will remain free. No premium tiers, no paywalls, no account required.

>
works offline

Once loaded, most tools continue working without internet. Built as progressive web apps.
