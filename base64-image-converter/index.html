Frequently Asked Questions

A data URI embeds binary data directly in the format data:image/png;base64,.... This lets you use images without a separate HTTP request — useful for inline CSS, HTML email, or JSON payloads.
Drag and drop an image onto the left panel or click to browse. The tool automatically generates the Base64 data URI, which you can copy or download as a text file.
Base64 encoding increases file size by about 33%. For small images (icons, logos under 10KB), this is fine. For larger images, use a separate file or CDN.
