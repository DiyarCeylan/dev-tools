A data URI is a way to embed binary data (like images) directly in HTML, CSS, or JavaScript. It starts with data:image/png;base64, followed by the Base64-encoded data. This tool extracts and displays the original image from that string.
You can get a data URI by converting an image to Base64 using our Image to Base64 tool, or from browser dev tools, or exported from design tools like Figma or Sketch.
This tool is specialized for images. For decoding arbitrary Base64 strings to text or binary files, use our main Base64 Encoder / Decoder tool which handles all formats.
