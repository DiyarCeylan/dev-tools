Frequently Asked Questions

HEX (#ff6600 or #f60), RGB (rgb(255,102,0)), RGBA (rgba(255,102,0,0.5)), HSL (hsl(24,100%,50%)), HSLA, OKLCH (oklch(0.62 0.22 45)), and named CSS colors (red, coral, etc.).
OKLCH is a perceptually uniform color space. Unlike RGB or HSL, a change in OKLCH values produces a visually uniform change in color. It consists of lightness (L), chroma (C), and hue (H). This site uses OKLCH for its theme.
Named CSS colors are limited to 148 standard names. If your color doesn't have an exact match in the named palette, "—" is shown. The nearest named color is not approximated — only exact matches are reported.
