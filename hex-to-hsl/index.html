FAQ

Frequently Asked Questions

HSL stands for Hue, Saturation, Lightness. Hue is the color angle (0–360°), Saturation is the intensity (0–100%), and Lightness is the brightness (0–100%). It's a more intuitive way to describe colors than HEX or RGB.
Hue is measured in degrees on a color wheel: 0° = Red, 120° = Green, 240° = Blue. Values between represent all other colors. For example, 60° is yellow and 300° is magenta.
HSL is great for creating color schemes. You can keep the same hue and just adjust saturation/lightness for hover effects, gradients, and themes. CSS supports HSL natively with the hsl() and hsla() functions.
