FAQ
A Unix timestamp is the number of seconds (or milliseconds) since January 1, 1970, 00:00:00 UTC. It is timezone-independent.
Check if your timestamp is in seconds or milliseconds. The tool supports both. Milliseconds (13 digits) give a date far in the future if interpreted as seconds.
On 32-bit systems, timestamps will overflow after January 19, 2038. Most modern systems use 64-bit integers, so this is rarely an issue today.
