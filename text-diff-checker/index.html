Frequently Asked Questions

This tool uses a line-based LCS (Longest Common Subsequence) diff algorithm. It compares texts line by line, identifying unchanged, added, and removed lines. Use the Ignore case and Ignore whitespace options to normalize comparisons.
The tool runs entirely in your browser with no file size limit. Very large files (10000+ lines) may cause the diff algorithm to take a few seconds. Smaller files diff instantly.
